NEROP Installer

Проект NEROP (NERve OPtic) - установщик и набор утилит для работы с оптическими нервами.
Структура проекта
text

nerop-installer/      - Основная папка с установщиком и утилитами

Установка
Быстрая установка:
curl -sSL https://raw.githubusercontent.com/nerop-nerop/nerop-installer/main/nerop-installer/nerop-installer.py | python3

git clone https://github.com/nerop-nerop/nerop-installer.git
cd nerop-installer, и после того как перешли ещё раз пишете cd nerop-installer

Запуск установщика:
bash

cd nerop-installer
./nerop-installer.py

Возможности

    Автоматическая установка всех компонентов

    Настройка окружения

    Управление оптическими нервами

    Диагностика и мониторинг

Требования

    Linux (рекомендуется Arch Linux или производные)

    Python 3.x

    Стандартные системные утилиты

Поддержка

При возникновении проблем:

    Проверьте логи установки

    Убедитесь в наличии прав доступа

    Проверьте подключение к интернету
